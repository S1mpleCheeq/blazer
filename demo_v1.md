# 智能体通信网关Demo1.0展示对接
## Demo1:已知地图导航巡检
### 功能描述
在已知地图里，发布巡检任务(定时在某几个仓库巡检，识别人员)，拆解完返回规划任务点序列，机器人平台规划路线，端侧自主巡航
### 穿插功能点
1. 中途没电回充电桩，充满电回去继续巡逻
2. 暂时中断当前任务，指定去某个任务点去检查，检查完毕、继续巡航
### 硬件平台提供信息
1. 能提供的动作：走路 拍照
2. 地图文件，点云地图(没有矢量信息)，带标注物体的语义信息的坐标 (示例 :  json : 办公室A:(1,2.0), 仓库B:(2,4,0) )
3. 当前机器的位置 消息格式ros::odometry
4. 设备电量信息
5. 系统：Ubuntu20.04 硬件nx
### 智能体网关输出
1. 任务点三维坐标序列和语义信息(json)，语义信息作为平台展示
